# План реализации Landing Page

## Overview

Создание современного landing page с тремя основными блоками на одной странице:
- **Hero** — главный блок с привлекательным заголовком и CTA
- **Content** — блок контента с информацией (аккордеон)
- **CTA** — блок для захвата контактных данных (форма)

**Технологический стек:**
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- shadcn/ui компоненты
- Aceternity UI компоненты
- shadcnblocks компоненты
- Framer Motion для анимаций

## Pages

### Главная страница (`/`)
- **Путь:** `src/app/page.tsx`
- **Описание:** Одностраничный landing page с тремя секциями
- **Структура:**
  - Hero Section (полноэкранный блок)
  - Content Section (блок с контентом)
  - CTA Section (форма захвата контактов)

## Layouts

### RootLayout
- **Файл:** `src/app/layout.tsx` (уже существует)
- **Описание:** Корневой layout с провайдерами тем и базовыми настройками
- **Компоненты:**
  - ThemeProvider
  - Toaster (уведомления)
  - Progress (индикатор прокрутки)

### LandingLayout (опционально)
- **Файл:** `src/components/layout/landing-layout.tsx` (новый)
- **Описание:** Специализированный layout для landing page
- **Функции:**
  - Управление секциями
  - Плавная прокрутка между секциями
  - Responsive поведение

## Step-by-step Tasks

### Этап 1: Подготовка компонентов
- [ ] **1.1** Установить компонент `textarea` из @shadcn
  ```bash
  npx shadcn@latest add textarea
  ```
- [ ] **1.2** Установить компонент `contact2` из @shadcnblocks
  ```bash
  npx shadcn@latest add @shadcnblocks/contact2
  ```
- [ ] **1.3** Проверить наличие всех зависимостей в `package.json`

### Этап 2: Создание компонентов секций
- [ ] **2.1** Создать `src/components/landing/hero-section.tsx`
  - Использовать `HeroHighlight` и `Highlight` из `@aceternity`
  - Добавить заголовок, подзаголовок и CTA кнопку
  - Реализовать адаптивный дизайн
  - Добавить анимации через Framer Motion

- [ ] **2.2** Создать `src/components/landing/content-section.tsx`
  - Использовать `Accordion` компоненты из `@shadcn`
  - Создать секции с вопросами/ответами или информацией
  - Добавить иконки и стилизацию
  - Реализовать плавные анимации раскрытия

- [ ] **3.3** Создать `src/components/landing/cta-section.tsx`
  - Использовать `contact2` из `@shadcnblocks` или создать кастомную форму
  - Поля: имя, email, сообщение (опционально: телефон, тема)

   - Добавить градиентный текст через `Highlight`
