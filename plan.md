# План разработки приложения для работы с документами

## Overview
Создание современного веб-приложения для управления документами с красивым UI. Приложение будет включать страницы логина, дашборда со списком документов и страницу просмотра документа.

## Страницы

### 1. Страница логина (`/login`)
- **Назначение**: Аутентификация пользователей
- **Layout**: `AuthLayout`
- **Компоненты**:
  - `@shadcn/card` - контейнер формы
  - `@shadcn/input` - поля ввода email/пароля
  - `@shadcn/button` - кнопка входа
  - `@shadcn/label` - подписи полей
  - `@aceternity/background-beams` - анимированный фон
  - `@cult-ui/texture-card` - карточка формы с текстурой

### 2. Страница дашборда (`/dashboard`)
- **Назначение**: Список всех документов пользователя
- **Layout**: `DashboardLayout`
- **Компоненты**:
  - `@shadcn/sidebar` - боковая навигация
  - `@shadcn/table` - таблица документов
  - `@shadcn/input` - поиск документов
  - `@shadcn/button` - действия с документами
  - `@shadcn/badge` - статусы документов
  - `@shadcn/pagination` - пагинация
  - `@cult-ui/minimal-card` - карточки документов
  - `@aceternity/spotlight` - эффект подсветки

### 3. Страница документа (`/documents/[id]`)
- **Назначение**: Просмотр и чтение документа
- **Layout**: `DocumentLayout`
- **Компоненты**:
  - `@shadcn/scroll-area` - область прокрутки документа
  - `@shadcn/button` - кнопки действий
  - `@shadcn/breadcrumb` - навигация
  - `@shadcn/tabs` - вкладки (если нужно)
  - `@cult-ui/texture-overlay` - текстурный фон
  - `@aceternity/tracing-beam` - эффект чтения

## Layout'ы

### 1. RootLayout
- **Компоненты**:
  - `@shadcn/sonner` - уведомления
  - `@shadcn/progress` - индикатор загрузки

### 2. AuthLayout
- **Компоненты**:
  - `@aceternity/background-beams` - анимированный фон
  - `@cult-ui/bg-animated-gradient` - градиентный фон

### 3. DashboardLayout
- **Компоненты**:
  - `@shadcn/sidebar` - боковая панель
  - `@shadcn/navigation-menu` - главное меню
  - `@shadcn/avatar` - аватар пользователя
  - `@cult-ui/dock` - док-панель

### 4. DocumentLayout
- **Компоненты**:
  - `@shadcn/scroll-area` - основная область
  - `@shadcn/toolbar` - панель инструментов
  - `@cult-ui/floating-panel` - плавающая панель

## Компоненты по страницам

| Страница | shadcn/ui | Aceternity UI | Cult UI |
|----------|-----------|---------------|---------|
| `/login` | card, input, button, label | background-beams | texture-card |
| `/dashboard` | sidebar, table, input, button, badge, pagination | spotlight | minimal-card |
| `/documents/[id]` | scroll-area, button, breadcrumb, tabs | tracing-beam | texture-overlay |

```
│   │   ├── dashboard-layout.tsx  # DashboardLayout
│   │   └── document-layout.tsx   # DocumentLayout
│   ├── ui/                       # shadcn/ui компоненты
│   ├── auth/
│   │   └── login-form.tsx        # Форма логина
│   ├── dashboard/
│   │   ├── documents-table.tsx   # Таблица документов
│   │   └── search-bar.tsx        # Поиск документов

│   └── constants/

```
